# 📂 Sistema de Administración de Expedientes
Sistema integral desarrollado en **.NET (C#)** para la gestión eficiente de trámites, con un enfoque en escalabilidad y mantenibilidad.

## 🛠️ Stack Tecnológico
* **Frontend:** Blazor (Interfaz reactiva y moderna).
* **Backend:** .NET Core con **Arquitectura Limpia (Clean Architecture)**.
* **Persistencia:** SQLite con **Entity Framework Core**.
* **Patrones:** Inyección de Dependencias y Autenticación de usuarios.

## 🚀 Características Principales
* Gestión completa del ciclo de vida de expedientes.
* Sistema de login.
* Código organizado por capas para facilitar el testeo y mantenimiento.
